### About

Bahrain.js is where JavaScript developers in Bahrain come together — to **build**, **ship**, **learn**, and **connect**. We are a builder community, not a passive audience. Every meetup includes hands-on time. Every project targets a real `npm publish`. We measure output, not attendance.

### What we do

- 🎤 **Meetups** — Monthly talks + live demos
- 🛠️ **Builder Sessions** — Hands-on coding around a theme
- 🏗️ **Hackathons** — Quarterly weekend builds
- ☕ **Coffee & Code** — Informal co-working, no agenda
- 📦 **Open Source** — Community packages published under [`@bahrainjs`](https://www.npmjs.com/org/bahrainjs)

### How ideas become packages

```
Idea → Builder Session → Prototype → bahrain-js/repo → npm publish @bahrainjs/package
```

### Get involved

Anyone can start contributing — no permission needed:

| Path | How |
|---|---|
| **Attend** | Join a meetup or builder session |
| **Contribute** | Open a PR on any `bahrain-js` repo |
| **Speak** | Give a lightning talk at a meetup |
| **Write** | Submit a blog post or guide |
| **Build** | Propose a project for a builder session |

### Guiding principles

| Principle | What it means |
|---|---|
| **Output over activity** | We measure what members build and ship |
| **Low floor, high ceiling** | Anyone can join — the path to maintainer is transparent |
| **Local first, open always** | Rooted in Bahrain, open source to everyone |
| **Practical over theoretical** | Talks include demos, workshops produce working code |
